MASTERMIND

MASTERMIND RULES

Mastermind is a game of skill and logic that consists of discovering the hidden sequence of colors.

ELEMENTS THAT MAKE UP THE GAME

Board composed of:
• 8 rows of large holes for the colored spheres.
• 8 rows of small holes for the white and red clues.
• 4 holes to hide the secret code with the shield.

HOW TO PLAY

In Mastermind you can play with one or two players, you will create a hidden code with 4 colored spheres, you can make combinations with the 8 available colors and even repeating color if you wish. The color code will be hidden with the shield so you can not see it, you must hit in the fewest possible moves the key to get a good score. To decipher the secret color code the player must try random combinations of colors, and in each combination, you will have clues through the white and red spikes. For each correct sphere in color and position, a red spike will appear, and for each correct color but in the wrong place, a white spike will appear.

SCORING

The way of scoring is quite simple: it can be agreed before starting how many games the game will be. At the end of each game, the player who has created the code will earn as many points as rows the opponent has covered until he has managed to crack the code. That is to say, if he needed to try 7 different combinations before getting it right, this is how many points the opposing player earns.
